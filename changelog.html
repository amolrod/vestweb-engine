Changelog

All notable changes to Vestweb Engine will be documented in this file.

v2.0.0 November 19, 2025
New Features
  • Physics Engine: Real-time rigid body physics with sphere-sphere and sphere-plane collisions.
  • Advanced Controls: Full WASD keyboard support, Pointer Lock mouse control, and Gamepad support.
  • PBR Rendering: Complete Cook-Torrance BRDF implementation with GGX distribution.
  • Post-Processing: Added SSAO, Bloom, Volumetric Fog, and ACES Tone Mapping.
  • Virtual FS: In-memory file system for asset management.
Performance
  • Optimized WebGL 2.0 pipeline for macOS (Apple Silicon) and Windows.
  • Adaptive resolution scaling based on device capabilities.
  • Efficient shader compilation and uniform caching.
v1.0.0 October 2025
Initial Release
  • Basic Path Tracing implementation.
  • Simple sphere rendering.
  • Initial WebGL setup.
