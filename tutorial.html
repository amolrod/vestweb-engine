Interactive Tutorial

Learn how to master Vestweb Engine in 5 simple steps.

Step 1

Launch the Engine

Open motor.html in your browser. You'll see the loading screen followed by the main scene.

[Screenshot: Loading Screen]
Step 2

Basic Controls

Navigate the 3D world using standard FPS controls:

  • WASD to move horizontally
  • Space to fly up
  • Shift to fly down
  • Mouse to look around (Click to lock cursor)
Step 3

Developer Console

Open your browser's developer tools (F12 or Cmd+Opt+I). Type the following command to verify the engine is running:

VESTWEB.info()

You should see a formatted output with version and system info.

Step 4

Creating Physics Objects

Let's spawn a bouncing ball! Paste this code into the console:

VESTWEB.addPhysicsObject({
    position: [0, 10, 0],
    velocity: [0, 0, 0],
    mass: 1.0,
    radius: 0.5,
    restitution: 0.9, // High bounce
    friction: 0.5,
    dynamic: true
});

Look up! You should see a sphere falling and bouncing.

Step 5

Advanced Physics

Try creating a heavy bowling ball:

VESTWEB.addPhysicsObject({
    position: [2, 5, 0],
    velocity: [0, 0, 0],
    mass: 10.0,       // Heavy
    radius: 0.8,
    restitution: 0.1, // Low bounce
    friction: 0.8,
    dynamic: true
});
