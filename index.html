Engineered for Performance

Built from the ground up to leverage modern GPU capabilities without the bloat.

PBR Rendering

Physically Based Rendering pipeline with Cook-Torrance BRDF, Fresnel-Schlick, and GGX distribution for realistic materials.

Path Tracing

Real-time path tracing with up to 5 bounces, progressive accumulation, and Russian Roulette optimization.

Physics Engine

Integrated Verlet integration physics system with sphere-sphere/plane collisions and configurable gravity.

Post-Processing

Cinematic effects stack including SSAO, Bloom, Volumetric Fog, and ACES Tone Mapping.

Cross-Platform

Optimized for macOS (M1/M2/M3), Windows, and Linux. Adaptive resolution scaling for best performance.

Zero Dependencies

Pure vanilla JavaScript. No frameworks, no build tools required. Just drop in and render.
